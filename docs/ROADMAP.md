# Roadmap

The active restructuring plan is tracked in [PORTAL_RESTRUCTURE.md](PORTAL_RESTRUCTURE.md). The long-term product blueprint is tracked in [INTERNAL_PORTAL_BLUEPRINT.md](INTERNAL_PORTAL_BLUEPRINT.md). This roadmap stays as the delivery checklist.

## Phase 0: Mobile and Access Foundation

- Replace the mobile sidebar with a bottom navigation and role-aware More menu.
- Keep route/controller permission checks aligned with visible menus.
- Validate Employee, HR, IT, Admin, and executive-style access on desktop and mobile.

## Phase 1

- Login with employee code and password.
- Employee profile.
- Announcements.
- Knowledge videos and articles.
- Payroll redirect.
- Document download prototype.
- Employee Directory with branch, department, position, status, search, profile, assigned asset, and access-right readiness.
- Basic Ticket / Complaint & Request flow with category, priority, owner, status, attachments, SLA fields, and comments.
- HR to IT onboarding workflow with manager approval readiness, IT checklist, access items, and asset assignment.
- Basic access request records for new/change/remove access.
- IT Asset register with asset code, holder, status, warranty date, category, export, and audit trail.
- Basic executive dashboard cards for tickets, onboarding, assets, warranty, pending approvals, and SLA.
- Audit log for create, update, approval, export, file, access, and asset actions.

## Phase 2

- Helpdesk workflow.
- Notifications.
- IT dashboard filters and assignments.
- SmartFlow Work Center with CSV import, assignee/SLA updates, comments, and URL attachments.
- IT asset management inside WDC.
- CEO/executive dashboard draft with read-only metrics.
- Replace URL attachments with first-party file storage when storage policy is confirmed.
- Employee offboarding with manager confirmation, IT disable checklist, asset return, and employee status change.
- License management and license expiry alerts.
- Warranty alerts at 30, 60, and 90 days.
- Approval Center for onboarding, access requests, IT requests, asset transfer, and offboarding.
- Announcement read tracking, audience targeting, and expiry.
- Knowledge Base publishing workflow and FAQ.
- Report dashboard for Ticket, Asset, Employee, Access, SLA, and Audit Log exports.

## Phase 3

- Complaint case management.
- Knowledge Base publishing workflow.
- Statistics reports.
- Permission policy layer and finer role rules.
- Full SmartFlow retirement plan: final exports, data-owner approval, and read-only legacy archive.
- AD, Google Workspace, Microsoft 365, SAP, CRM, and POS metadata integration.
- QR Code asset labels and scan-to-detail flow.
- E-Sign / digital acknowledgement for asset handover and policy acknowledgement.
- Advanced BI dashboard for executives.
